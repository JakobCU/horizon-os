# Past CORDIS Abstracts — Reference Database

> **Purpose**: Repository of relevant past EU-funded project abstracts for the Consortium Scout and Excellence Architect to reference when identifying partners, positioning the state of the art, and demonstrating awareness of the EU research landscape.
> **Read by**: Consortium Scout, Excellence Architect, Call Strategist

## How to Use This File

1. Search CORDIS (https://cordis.europa.eu) for projects relevant to your call topic.
2. Paste abstracts below, organized by thematic cluster.
3. The Consortium Scout uses this to find organizations with proven EU project track records.
4. The Excellence Architect uses this to position the proposal relative to prior/ongoing work.
